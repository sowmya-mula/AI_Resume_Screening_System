# 🤖 AI Resume Screening Dashboard

An end-to-end **AI-powered Resume Screening System** that automates candidate evaluation using **Natural Language Processing (NLP), Machine Learning, SQL analytics, and Power BI dashboards**.

This project processes resumes, extracts skills, calculates candidate-job similarity scores, ranks applicants, and provides interactive recruiter insights through a professional dashboard.

---

# 📌 Project Overview

Traditional resume screening is time-consuming and difficult to scale.

This project builds an automated pipeline that:

✅ Processes resumes using NLP  

✅ Extracts candidate skills  

✅ Computes similarity scores using ML 

✅ Ranks candidates automatically  

✅ Stores analytics using SQL  

✅ Visualizes recruiter insights in Power BI
